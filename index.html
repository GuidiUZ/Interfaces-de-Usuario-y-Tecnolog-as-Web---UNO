INTERFACES DE USUARIO Y TECNOLOGIAS WEB - UNO

Git (El Software) Git es un sistema de control de versiones que instalas en tu propia computadora. Función: Lleva un registro de todos los cambios que haces en tus archivos. Es como una "máquina del tiempo" que te permite volver a versiones anteriores si algo sale mal. Ubicación: Vive localmente en tu PC. No requiere internet: Puedes hacer cómics (guardar estados) sin estar conectado. 2. GitHub (La Plataforma) GitHub es una plataforma en la nube que aloja proyectos que usan Git. Función: Sirve para subir tus proyectos (repositorios) a internet para que otros puedan verlos, colaborar contigo o simplemente para tener un respaldo. Ubicación: Es un sitio web (propiedad de Microsoft). Extra: Ofrece herramientas sociales como perfiles, foros de discusión y gestión de tareas que Git por sí solo no tiene.
